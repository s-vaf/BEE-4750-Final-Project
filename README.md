# BEE-4750 Final Project

## Overview
This is the template for the BEE-4750 Final Project.

## Project Structure
```
├── Project.toml          # Julia project manifest
├── src/                  # Source code directory
│   └── BEE4750FinalProject.jl
├── notebooks/            # Jupyter notebooks
├── data/                 # Input/output data
└── README.md
```

## Setup
1. Open Julia
2. Activate the project: `] activate .`
3. Add dependencies as needed: `] add Package`

## Usage
```julia
include("src/BEE4750FinalProject.jl")
using .BEE4750FinalProject
hello_world()
```
